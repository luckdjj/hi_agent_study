## 📋 每日任务要求

### 通用要求

- 完成对应章节的阅读
- 完成课后习题或输出阶段笔记
- 保留关键截图或运行结果
- 提交一段自己的学习心得

### 特殊要求

- **Task00**：跑通第一章代码
- **Task01**：保留快速构建智能体的关键截图
- **Task03**：保留本地模型跑通截图
- **Task04**：保留代码跑通截图
- **Task05**：保留任意一个低代码平台的实操截图
- **Task06**：完成一个不同于教程示例的个人 demo

## 📂 建议文件使用方式

- `README.md`：查看仓库整体说明
- `打卡总表.md`：汇总每个 Task 的状态和交付物
- `项目总结.md`：阶段结束后做一次复盘
- `TaskXX/README.md`：查看每个 Task 的任务清单
- `TaskXX/ToStudyList.md`：记录当天学习内容、实践过程和心得
- `TaskXX/截图/`：统一保存运行截图和结果截图

## 🤖 与大模型一起共学

- 可以让你自己的大模型先帮你拆解每个 Task 的学习顺序
- 学习过程中，让它根据 `ToStudyList` 追问你当前进度、问题和下一步
- 学习结束后，让它帮你整理反馈、生成一段更清晰的学习总结
- 模型更适合做陪练和复盘助手，核心实践过程仍建议自己亲手完成

## 🎯 学习目标

1. **理论基础**：掌握智能体的核心概念和发展历史
2. **技术原理**：理解大语言模型和 Transformer 架构
3. **范式实践**：实现 ReAct、Plan-and-Solve、Reflection 等经典范式
4. **平台体验**：熟悉 Coze、Dify、n8n 等低代码平台
5. **框架应用**：掌握主流智能体开发框架的使用
